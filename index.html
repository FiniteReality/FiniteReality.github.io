disccord

disccord is a C++ library for the popular chat platform, Discord.

Right now, disccord is not in a stable state but it is being worked on. When it is complete, it will be a cross-platform solution to the lack of well-designed and implemented C++ libraries.

(GitHub)

Discord.Net

Discord.Net is another library, this time for C#, for Discord.

I have contributed a lot towards the commands extension for the latest version, Discord.Net 1.0.
